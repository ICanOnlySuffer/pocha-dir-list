# PTree (pocha-tree)

A simple directory visualizer based on 'tree'.

## Installation

```shell
$ git clone https://github.com/ICanOnlySuffer/ptree
$ cd ptree
# ./install
```

## Uninstallation

```shell
# ./uninstall
```
